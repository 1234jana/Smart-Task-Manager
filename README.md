# 🧠 Smart Task Manager

## 📌 Project Overview

Smart Task Manager is a modern web-based application built using React that helps users efficiently manage their daily tasks. The application provides full CRUD functionality along with filtering options and persistent storage using localStorage, ensuring a smooth and user-friendly experience.

---

## 🛠️ Technologies Used

* ⚛️ React.js
* 💻 JavaScript (ES6+)
* 🎨 HTML & CSS
* 💾 localStorage

---

## ✨ Features

* ➕ Add new tasks
* ✏️ Edit existing tasks
* ❌ Delete tasks
* 🔍 Filter tasks (All / Completed / Pending)
* 💾 Persistent data storage using localStorage
* 🎨 Colorful and responsive UI/UX design

---

## 📂 Project Structure

* `src/` – Application logic and components
* `public/` – Static files
* `App.js` – Main component
* `App.css` – Styling

---

## ▶️ How to Run the Project

1. Clone the repository
2. Navigate to the project folder
3. Install dependencies:

   ```bash
   npm install
   ```
4. Start the application:

   ```bash
   npm start
   ```
5. Open in browser:

   ```
   http://localhost:3000
   ```
